# app08_StackNav_03

Projeto Expo/React Native com Stack Navigation.

Tema: Curitiba Explorer.

Telas:
- Principal
- Pontos Turísticos com FlatList
- Roteiro do Dia com Switch
- Preferências com Picker

## Como executar

```bash
npm install
npx expo start
```

Este projeto já inclui a dependência `expo` no package.json para evitar o erro:
`Cannot determine the project's Expo SDK version because the module expo is not installed.`
